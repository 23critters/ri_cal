                                <div class="method-description">

                                        <p>

Set the default tzid to be used when instantiating an instance from a ruby object see RiCal::PropertyValue::DateTime.from_time

Parameter

value:A string value to be used for the default tzid, a value of &8216;none&8217; will cause

values with NO timezone to be produced, these are interpreted by iCalendar as floating times i.e. they are interpreted in the timezone of each client. Floating times are typically used to represent events which are &8216;repeated&8217; in the various time zones, like the first hour of the year.
